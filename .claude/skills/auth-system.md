# 账号与权限系统

## 角色定义

| 角色 | 权限 |
|------|------|
| `root` | 管理员，可创建/禁用账号，查看和操作所有数据 |
| `tester` | 测试人员，参与评测和评审，只能查看分配给自己的任务 |

## 权限矩阵

| 操作 | root | tester |
|------|:----:|:------:|
| 创建新账号 | ✅ | ❌ |
| 创建客户/应用/模板 | ✅ | ❌ |
| 创建实验 | ✅ | ✅ |
| 录入客观指标 | ✅ | ✅ |
| 进行主观评测 | ✅ | ✅ |
| 查看所有数据 | ✅ | 仅相关 |

## 默认账号

- 用户名: `root`
- 密码: `root123`
- 首次登录应修改密码
